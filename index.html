SaiKanji (彩漢字)

A color stroke-order font for kanji.

SaiKanji turns KanjiVG stroke data into a modern hybrid OpenType font with per-stroke color, stroke-order numbers, optional guide grids, and broad app/browser compatibility.

Install

Install once, then use it like a normal font.

Desktop

  1. Download a .ttf file from the GitHub release.
  2. Open the file and install it with your system font manager.
  3. Select the SaiKanji style in Pages, design tools, Anki, or other apps.

Web

Use the WOFF2 files with @font-face. Host the files with your site when possible so font loading is cacheable and predictable.

Files

One-click TTF downloads.

Gradient files preserve smooth stroke color. Solid files use flat stroke color. Black files are the simplest monochrome fallback.
